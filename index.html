Kastanza

Kastanza App

×
Summary: The purpose of this application is to help people find clean bathrooms throughout the city. Users can rate bathrooms by giving using the star rating system.
Technologies Used: HTML5, CSS3, SASS, Ruby, Sinatra, SEQUEL ORM, SQL, Javascript, jQuery, PostgreSQL(Production), SQLite(Development), Heroku
Project Status: Alpha (WIP)
Unsolved Problems:
  • Currently the average number of stars is calculated on the fly when the server send the data back to the client side. A better solution would be to cache data for faster access using Redis or Firebase.
  • Some small unresolved JS issues where selectors are trying to select items that do not exist. At this moment they do not effect functionality.

ChiStreetArt

ChiStreetArt App

×
Summary: Just like any other world class city, Chicago offers an amazing array of art that can be seen in a museum or on a brick wall. A ride up and down the L will only expose you to a fraction of the Chicago street art on display throughout the city. This page is intended to lessen that gap. It is our hope to catalogue the history of installations and commissioned walls through crowd sourced entries, allowing you to follow the transition of these communal art spaces. We can’t always be the eyes on the street and need your help to allow this project to function. We’d love to see what moves and inspires you. Send us your photos.
Technologies Used: HTML5, CSS3, LESS, Node, Express.js, Handlebars, MongoDB, Javascript, jQuery, BCrypt, Heroku, Digital Ocean(Ubuntu Linux), MLabs
Project Status: Alpha (WIP)
Unsolved Problems:
  • Add image carousel for each location.
  • Issues with image sizing and resizing.
  • Betteer efficiency with the API server.
  • Select a Javascript Framework like React/Angular.
  • Create a mobile application for the app.
